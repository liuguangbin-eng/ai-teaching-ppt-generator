# Chinese Teaching PPT Generator

An AI-powered web tool that generates structured Chinese teaching PPT slides for foreign students (HSK 3–5 level).

## Features

- **AI-generated lesson slides** — input a topic or vocabulary list, get a complete PPT structure
- **Structured slide types** — title, vocabulary, sentence patterns, grammar, practice, application, summary
- **Vocabulary details** — Chinese characters, pinyin, meaning, example sentences, and image prompts
- **Interactive exercises** — multiple choice, fill-in-the-blank, speaking tasks, matching
- **Real-life dialogues** — practical scenario-based conversation slides
- **Export options** — download as Markdown or `.pptx` file
- **Bilingual support** — English or Indonesian translations
- **Adjustable difficulty** — HSK 3, HSK 4, or HSK 5
